Trimmy — Paste once, run once

Trimmy flattens multi‑line shell snippets you copy so they paste and execute cleanly. Two menu actions: Paste Trimmed (with “X trimmed” badge) and Paste Original.

Download latest release
macOS 15+ Swift 6 · Sparkle updates Menu bar, no Dock icon

What it does

  • Detects shell-like copies and flattens newlines (respects \ continuations).
  • Strips prompt gutters and decorative box glyphs (│) that terminal UIs inject.
  • Shows “X trimmed” on Paste Trimmed; Paste Original leaves your copy untouched.
  • Three aggressiveness levels + toggles for blank lines and box stripping.
  • Global hotkeys for both trimmed and original paste paths.

Why it’s handy

  • Run docs/ChatGPT snippets without hand-editing line wraps.
  • Fixes terminal-wrapped URLs and pipelines that pick up stray box characters.
  • Menu previews show target app and what will be pasted.
  • Sparkle updates keep you current; launch-at-login available.
